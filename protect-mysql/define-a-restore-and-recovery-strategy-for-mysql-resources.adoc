= MySQL 리소스에 대한 복원 및 복구 전략 정의
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
데이터베이스를 복원하고 복구하기 전에 전략을 정의해야 복원 및 복구 작업을 성공적으로 수행할 수 있습니다.


NOTE: 데이터베이스의 수동 복구만 지원됩니다.

.단계
. 수동으로 추가된 MySQL 리소스에 대해 지원되는 복원 전략을 확인합니다.
. 자동 검색된 MySQL 데이터베이스에 지원되는 복원 전략을 확인합니다.
. 수행할 복구 작업 유형을 결정합니다.
