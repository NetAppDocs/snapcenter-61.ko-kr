= 동일한 볼륨에 여러 데이터베이스가 있음
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
백업 정책에 백업당 최대 데이터베이스 수를 설정하는 옵션이 있으므로(기본값은 100) 모든 데이터베이스를 동일한 볼륨에 넣을 수 있습니다.

예를 들어, 동일한 볼륨에 200개의 데이터베이스가 있는 경우 두 개의 스냅샷이 생성되고, 두 개의 스냅샷에는 각각 100개의 데이터베이스가 포함됩니다.
