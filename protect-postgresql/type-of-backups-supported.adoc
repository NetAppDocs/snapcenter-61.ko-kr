== 스냅샷 복사 기반 백업

스냅샷 복사 기반 백업은 NetApp 스냅샷 기술을 활용하여 PostgreSQL 클러스터가 있는 볼륨의 온라인, 읽기 전용 복사본을 만듭니다.
